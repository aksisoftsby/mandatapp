Apakah Mandat-APP

Mandat adalah singkatan dari kata Management Data. Aplikasi mandat berfokus pada pengolahan data yang simple dan memiliki banyak fitur dan keuntungan bagi Pengelola dan non pengelola / Crowdsource sebagai tahap memviralkan data dan memberikan hak akses yang bisa dibagi dan digunakan Netizen.

Fitur utama Mandat-App

Mandat-app dilengkapi dengan fitur yang terus dikembangkan dan dikelola agar sesuai dengan trend dan tekhnologi serta sistem terbaru. Saat ini fitur fitur utama Mandat-app adalah :

  1. Auto Generate API dan dokumentasi:

    Setiap modul yang sudah ada akan otomatis terbuat data dan dokumentasinya. Dengan data yang dinamis yaitu ketika pengelola merubah/mengupdate data maka output API juga akan berubah sesuai dengan data yang dimasukkan. Format dan Key yang digunakan tetap sama, sehingga para developer aplikasi yang memerlukan Open Data berupa API dapat mengintegrasikan aplikasi mereka dengan lebih yakin. Dengan menggunakan Open Data API, menjadikan data anda lebih transparansi dan terbuka untuk umum.

  2. Embed widget:

    Bagi Website Developer, Blogger, Android Developer mendapatkan informasi dari website anda bisa menjadi sangat penting terutama rekanan, serta lembaga yang terkait bahkan blogger/developer yang tertarik mencantumkan widget website pengelola kedalam aplikasi/website yang mereka kelola. Keuntungan utama memfasilitasi sebuah app dengan widget adalah menjadikannya lebih viral karena tentu saja jika Blogger/Developer tersebut mempunyai website/aplikasi yang sudah viral tidak menutup kemungkinan Embed Widget dari aplikasi pengelola dapat digunakan dan dikunjungi Netizen tersebut.

  3. Mudah digunakan dengan tutorial lengkap:

    Dengan berbagai modul dan halaman pada Mandat-app tidak menutup kemungkinan anda akan menemukan pengalaman menggunakan metode web terbaru yang mungkin anda baru lihat dan kenal. Seperti google map, polygon code kecamatan, kode embed, memanipulasi template DPT, Menggunakan html table dan lain lain. Didalam setiap modul kami sematkan teks bantuan yang bisa anda baca dan pahami. Selain itu Kami juga menyediakan groups whatsapp dan telegram yang bisa anda gunakan untuk berbagi informasi, request fitur/modul dan berbagi pengalaman dengan pengguna Mandat-app lainnya.

Spesifikasi Server
